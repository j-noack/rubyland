## Installation

#### Supported Platforms
* Linux
* Windows
* OS X

#### Requirements
* [Ruby](https://www.ruby-lang.org/de/) (minimum supported version 2.2)
* [gosu] (https://libgosu.org/) (minimum supported version 0.10.4)

#### Grab Source
`git clone https://github.com/j-noack/rubyland.git`

#### Run
`cd rubyland.git && ruby main.rb`
